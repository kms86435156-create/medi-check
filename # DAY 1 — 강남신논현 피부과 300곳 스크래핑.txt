# DAY 3 — 로컬 MySQL DB 구축 및 데이터 일괄 삽입

## 목표
hospitals와 reviews 테이블을 설계하고, Day 1·2의 CSV/JSON 데이터를 SQLAlchemy bulk insert로 빠르게 적재합니다.

## 지시사항
1. `database/schema.sql` 파일을 만드세요:
   ```sql
   CREATE TABLE hospitals (
     id INT AUTO_INCREMENT PRIMARY KEY,
     name VARCHAR(100) NOT NULL,
     phone VARCHAR(20),
     address VARCHAR(200),
     hours VARCHAR(300),
     place_url VARCHAR(500),
     lat DOUBLE, lng DOUBLE,
     ai_summary JSON,
     premium_rank INT DEFAULT 0,
     created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   );
   CREATE TABLE reviews (
     id BIGINT AUTO_INCREMENT PRIMARY KEY,
     hospital_id INT NOT NULL,
     review_text TEXT,
     rating TINYINT,
     review_date DATE,
     FOREIGN KEY (hospital_id) REFERENCES hospitals(id)
   );
   ```
2. `database/seed.py` 파일: SQLAlchemy `bulk_insert_mappings`로 데이터 삽입
3. `.env` 파일에 DB 접속 정보 저장 (python-dotenv 사용)
4. 삽입 완료 후 카운트 검증 쿼리 출력

## 결과물
- `database/schema.sql`
- `database/seed.py`
- `.env.example`

# DAY 7 — LBS 위치기반 검색 API 개발

## 목표
사용자 위경도를 받아 반경 5km 내 병원을 AI 평점순으로 반환하는 API를 구현합니다.

## 지시사항
1. 카카오 로컬 API로 병원 주소 → 위경도 변환 (`utils/geocoding.py`):
   `GET https://dapi.kakao.com/v2/local/search/address.json`
2. `routers/search.py` 에 엔드포인트 추가:
   `GET /api/search?lat=37.5&lng=127.0&radius=5&sort=ai_score`
3. Haversine 공식 SQL 구현:
   ```sql
   SELECT *, (6371 * acos(cos(radians(:lat)) * cos(radians(lat))
   * cos(radians(lng) - radians(:lng)) + sin(radians(:lat))
   * sin(radians(lat)))) AS distance
   FROM hospitals HAVING distance < :radius
   ORDER BY distance ASC
   ```
4. 응답에 distance(km) 필드 포함
5. Postman 컬렉션 파일(`postman/medi-check.json`) 생성

## 결과물
- `routers/search.py`
- `utils/geocoding.py`
- Postman 컬렉션

## 지시사항

   ```dockerfile
   FROM python:3.11-slim
   WORKDIR /app
   COPY requirements.txt .
   RUN pip install -r requirements.txt
   COPY . .
   CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
   ```
2. `docker-compose.yml` 3개 서비스: fastapi, mysql, nginx
3. `nginx/default.conf`:
   - 80 → 443 리다이렉트

   - Upstream: `http://fastapi:8000`

   `certbot --nginx -d api.medi-check.com`
5. 배포 스크립트 `deploy.sh` 작성:
   `git pull && docker-compose down && docker-compose up -d --build`
